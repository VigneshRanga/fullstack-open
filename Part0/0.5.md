# Exercise 0.5: Single Page App

```mermaid
sequenceDiagram
    participant U as User
    participant B as Browser
    participant S as Server

    U->>B: Go to exampleapp/spa URL
    
    B->>S: GET /spa
    activate S
    S-->>B: HTML page
    deactivate S
    
    B->>S: GET /main.css
    activate S
    S-->>B: CSS styles
    deactivate S
    
    B->>S: GET /spa.js
    activate S
    S-->>B: JavaScript code
    deactivate S
    
    Note over B: Run JavaScript
    B->>S: GET /data.json
    activate S
    S-->>B: Notes data
    deactivate S
    
    B->>B: Build page with notes
    B-->>U: Show completed page
```
