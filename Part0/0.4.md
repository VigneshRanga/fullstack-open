# Exercise 0.4: New Note

```mermaid
sequenceDiagram
    participant User
    participant Browser
    participant Server

    Note over User, Server: User creates a new note in SPA

    User->>Browser: Types note content and clicks Save
    activate Browser
    
    Browser->>Server: POST /new_note_spa
    Note right of Browser: Content-Type: application/json<br>{content: "New note content"}
    activate Server
    
    Server-->>Browser: HTTP 201 Created
    Note left of Server: Response: {message: "note created"}
    deactivate Server
    
    Browser->>Browser: JavaScript updates note list dynamically
    deactivate Browser
    
    Browser-->>User: New note appears without page reload
```
