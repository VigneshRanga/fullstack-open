# Exercise 0.4: New Note

```mermaid
sequenceDiagram
    participant U as User
    participant B as Browser
    participant S as Server

    U->>B: Write note text in box

    S-->>B: 201 Created
    Note left of S: Confirm note saved
    deactivate S

    B-->>U: Show new note on page
```
